# 📊 Retail Sales Data Analysis

## 📌 Project Overview
This project focuses on analyzing retail sales data to extract meaningful business insights using **SQL, Excel, and Power BI**.  
The goal is to understand **sales performance, customer behavior, and regional trends** to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze sales trends over time (monthly & yearly)
- Identify top-performing categories, products, and regions
- Evaluate customer purchasing behavior
- Calculate key business KPIs such as AOV and sales contribution
- Build an interactive dashboard for visualization

---

## 🛠️ Tools & Technologies
- **Excel** → Data cleaning & preprocessing  
- **MySQL (SQL)** → Data storage & analysis  
- **Power BI** → Data visualization & dashboard creation

## 🔄 Project Workflow
1. **Data Collection**
   - Imported raw retail dataset  

2. **Data Cleaning (Excel)**
   - Handled missing values  
   - Fixed date formats  
   - Ensured consistent data types  

3. **Data Analysis (SQL)**
   - Created MySQL database and tables  
   - Performed 25+ SQL queries to extract insights  

4. **Data Visualization (Power BI)**
   - Built an interactive dashboard with filters and KPIs  

---

## 📊 Key KPIs Analyzed
- Total Sales  
- Total Orders  
- Average Order Value (AOV)  
- Monthly & Yearly Sales Trends  
- Sales Contribution by Category  
- Customer Segmentation  
- Shipping Duration Analysis  

---

## 📈 Key Insights
- **Technology category** contributes the highest revenue (~37%)  
- **Top customers and regions** generate ~40% of total sales  
- Sales show **monthly growth trends and seasonal patterns**  
- Some shipping modes have **longer delivery durations**  

---

## 📉 SQL Analysis Highlights
- Aggregations using `SUM()`, `COUNT()`, `GROUP BY`  
- Time-based analysis using `DATE_FORMAT()` and `YEAR()`  
- Growth calculations using window functions (`LAG()`)  
- KPI calculations like AOV and sales contribution  

---

## 📊 Power BI Dashboard Features
- KPI Cards (Total Sales, Orders, AOV)  
- Sales by Category & Sub-Category  
- Monthly Sales Trend (Line Chart)  
- Regional & State-wise Performance  
- Customer Segment Analysis  
- Interactive Filters (Region, Category, Customer)
